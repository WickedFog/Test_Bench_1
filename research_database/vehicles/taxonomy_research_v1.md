## Notes on Taxonomy Structure

- **Vehicle Type** = platform (what the vehicle IS)
- **Driving Style** = how it is driven (what you DO with it)
- Some types are also styles (Drift, Drag, Crawl) — these are flagged as **DUAL**
- All styles are user-selectable in the wizard regardless of vehicle type
- Comp variants of a style load class-specific components from the rulebook database

## Edge Cases & Pending Decisions

| Item | Issue | Decision |
|------|-------|----------|
| Lawn Mower Racing | Gemini flagged — likely false positive from separate search results | DROPPED — no evidence of organized RC lawn mower racing |
| Pulling Truck | Mostly custom built, limited retail | INCLUDED — long community history, serious enough to warrant support |
| 1/4 Scale Dragster | Same situation as pulling | INCLUDED as type — flag for community input on comp class details |
| Scale Trail comp | Does formal comp exist? | NOT CONFIRMED — currently listed as no formal comp. Community input needed. |
| Rally style on non-Rally cars | Is it rally driving if you're not in a rally car? | DECIDED: No — rally style tied to vehicle type more than other styles |
